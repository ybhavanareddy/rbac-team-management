# Team Management System (RBAC)

## 🚀 Overview

This is a MERN stack application that manages users, roles, and permissions using Role-Based Access Control (RBAC).

## 🧩 Features

* Create and view users
* Create roles with permissions
* View permissions assigned to roles
* Dynamic UI updates based on selected role

## 🛠 Tech Stack

* Frontend: React
* Backend: Node.js, Express
* Database: MongoDB

## 📌 API Endpoints

### Users

* POST /api/users
* GET /api/users
* DELETE /api/users/:id

### Roles

* POST /api/roles
* GET /api/roles

### Permissions

* POST /api/permissions
* GET /api/permissions

## ▶️ Run Locally

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
